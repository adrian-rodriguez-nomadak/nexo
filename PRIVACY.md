# Política de privacidad de Nexo

Última actualización: 14 de julio de 2026.

Nexo almacena en el dispositivo la información que registras sobre agenda,
recordatorios, finanzas, deudas y preferencias. Si inicias sesión y utilizas la
sincronización, estos datos se transmiten cifrados mediante HTTPS al servicio de
Nexo y se almacenan en su base de datos para mantenerlos disponibles entre tus
dispositivos.

La función de Inbox inteligente envía a OpenAI únicamente el texto que decides
interpretar. Ese texto se usa para identificar la acción solicitada y devolver
datos estructurados. No se envía automáticamente el resto del contenido local.

Nexo solicita acceso a notificaciones para mostrar recordatorios y avisos de
pagos. La biometría, cuando se activa, es validada por el sistema operativo; Nexo
no recibe ni almacena huellas o rasgos faciales. El PIN se guarda mediante el
almacenamiento seguro del dispositivo.

Puedes exportar, restaurar o borrar tus datos locales desde Ajustes. Para
solicitar acceso, rectificación o eliminación de datos sincronizados, contacta
al responsable de Nexo mediante el canal de soporte publicado con la aplicación.

Esta política debe publicarse en una URL accesible antes de distribuir Nexo en
una tienda de aplicaciones y debe completarse con el nombre legal, correo de
soporte y jurisdicción del responsable.
